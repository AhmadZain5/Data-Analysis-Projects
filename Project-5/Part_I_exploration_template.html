Bivariate Exploration¶

Code Walkthrough:¶

Below I attempt to associate each level of each feature in the first category with each level of each feature in the second category. Doing so, I want to classify each student's response for each level of a feature that corresponds with a single observation to see how the different combinations form the final conclusions. And in doing so, I'm effectively increasing the size of the subset worked with exponentially. This will no longer give the count of students for a given response, but rather the number of possible combinations that constitute a certain scenario and.

Steps I followed:

  • First I subset the dataset based on the two features of interest
  • Then, I restructure part of the subset corresponding to one feature of interest and keeping the other feature intact
  • I then restructure the second feature and keeping the newly structured feature intact, in order to account for all combination of responses.

Conclusions¶

There was a clear trend in how subjective norm affects a student's interest in math. This trend was less evident in the case of correlation between subjective norms and level of confidence in math for a given student. These conclusions derived from the bivariate exploration remained consistent in multivariate exploration when subjective norms had a more clear effect on interest in math than self-efficacy. The trend also remained the same when Math Anxiety was the third factor instead of self-efficacy.
